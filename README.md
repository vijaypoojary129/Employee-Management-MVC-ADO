# Employee Management System 

A single-page Employee Management application built with **ASP.NET MVC** and **SQL Server**. It demonstrates full CRUD operations using **AJAX**, **ADO.NET**, and **Stored Procedures**.

## üöÄ Features
- **Single Page Interface:** Add and Manage employees on the same view without page reloads.
- **AJAX Forms:** Asynchronous data submission and retrieval using jQuery.
- **File Upload:** Handles profile image uploads and server-side storage.
- **Stored Procedures:** All database logic (Insert, Update, Select) is handled via SQL Stored Procedures.
- **Dynamic UI:** Instant updates to the grid after saving/editing data.
